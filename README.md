# Getting started
## Installation
**1.** ```git clone https://github.com/cyberinventor/ls-discovery.git```
<br>
 **2.** ```cd ls-discovery```
<br>
**3.** ```bash install.sh```
<br>
## Uninstall
**1.** ```bash uninstall.sh```

## check for updates
**1.** ```bash update.sh```
